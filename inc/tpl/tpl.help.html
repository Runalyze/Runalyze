Runalyze

Runalyze ist ein vollständig konfigurierbares Lauftagebuch als Webanwendung - so ausführlich und detailverliebt wie kein anderes. Runalyze wird entwickelt von Hannes Christiansen und Michael Pohl.

Eine Übersicht der Features von Runalyze findest du auf runalyze.de. Über aktuelle Änderungen und nützliche Tricks informieren wir dich in unserem Blog.

Konfiguration

Alleinstellungsmerkaml von Runalyze sind die vielen Möglichkeiten, das Lauftagebuch an deine persönlichen Bedürfnisse anzupassen. In der Konfiguration findest du alle Einstellungen. Dort kannst du eigene Sportarten oder Trainingstypen definieren, bestimmen welche Trainingsdaten in der Übersicht angezeigt werden sollen, Parameter für die Rechenspiele festlegen und vieles mehr.

Browser Support

Runalyze is a modern application and therefore we want to use the most modern web techniques. We will not try to support any outdated browser versions.
  • jQuery 2.x does not support IE8 (or less)
  • Flot charts work with IE 6+, Chrome, Firefox 2+, Safari 3+ and Opera 9.5+
